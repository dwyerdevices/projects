# Sylph

The 10 year old has a project to come up with a 3D representation of a specific vocabulary
word: _sylph_. Scouring various model sites, and this resampled version of a statue scan from the [Stanford 3D Scanning Repository](http://graphics.stanford.edu/data/3Dscanrep/) seemed like a good fit. We modified
the model further, by removing most of the base below the feet.
